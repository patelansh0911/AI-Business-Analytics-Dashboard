# AI-Powered Business Analytics & Forecasting Dashboard

## What you get
- Interactive Streamlit dashboard (KPIs, trends, category/region analysis)
- Sales forecasting using a RandomForestRegressor baseline
- Synthetic dataset included: `business_sales_dataset.csv`

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Files
- `app.py` : Streamlit app
- `business_sales_dataset.csv` : dataset
- `requirements.txt`
